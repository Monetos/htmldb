📑 HTML-DB Bookmarklet

Mit diesem Bookmarklet kannst du auf jeder Claude-Artefakt-Seite mit einem Klick einen neuen Eintrag in deine Datenbank anlegen.

1. URL deiner HTML-DB eintragen

Trage hier die URL deiner GitHub-Pages-Datenbank ein – das Bookmarklet wird darauf zeigen.

2. Bookmarklet in die Lesezeichenleiste ziehen

Stell sicher, dass die Lesezeichenleiste sichtbar ist (Strg+Umschalt+B in Chrome/Edge). Dann ziehe den lila Button mit der Maus dorthin:

➕ HTML-DB Hinzufügen

Falls dein Browser den Drag-Drop blockiert, kannst du den Code auch per Rechtsklick kopieren und manuell ein Lesezeichen anlegen:

▸ Code zum Kopieren anzeigen
URL oben eintragen…

3. Workflow

  1. In einem Claude-Chat ein gutes Artefakt erstellt → Veröffentlichen klicken
  2. Auf der entstehenden claude.ai/public/artifacts/...-Seite das Bookmarklet anklicken
  3. Ein kleines Popup erscheint mit Titel und Tag-Eingabe
  4. Tags eintragen (z. B. klasse-10, mathematik) → Hinzufügen
  5. Eintrag öffnet sich direkt in deiner HTML-DB im neuen Tab

Alternative: Direkt aus Claude-Chat (statt Veröffentlichen)

Du kannst das Bookmarklet auch auf einer normalen claude.ai/chat/...-Seite anklicken – die URL des aktuellen Chats wird übernommen. Praktisch, wenn du dir Chats markieren willst (z. B. besonders gute Lernpfad-Generierungen für später).
